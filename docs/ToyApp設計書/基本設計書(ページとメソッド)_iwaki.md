### URLとアクションの種類

- Users

 |  URL |  アクション | httpメソッド |
 | ---- | ---- | ----|
 |  /users  |  index  | GET |
 |  /users/:id  |  show  | GET | 
 |  /users/new | new | GET |
 |  /users | create | POST |
 |　　/users/:id/edit　| edit | GET |
 |　　/users/:id 　| update | PATCH |
 |　　/users/:id 　| destroy | DELETE |
